# NBA Analytics and Team Management System

A web-based NBA analytics platform built with ASP.NET Core MVC and Entity Framework Core.

## Tech Stack
- ASP.NET Core MVC (.NET 10)
- Entity Framework Core
- Microsoft SQL Server
- Bootstrap 5

## Features
- League & Conference Standings
- Team Payroll Rankings
- Top 10 Players by Points, Assists, Rebounds, Steals, Blocks, Salary
- Admin Panel with Match Management
- Automatic TeamStats updates via SQL Triggers

## Database
- 10 Tables, 12 Triggers, 12 Views, 10 Indexes
- 30 Teams, 300 Players, 150 Matches, 3000 Player Stats
- Real 2025-26 NBA data
